Simple Calculator (Python)

This is a beginner-friendly calculator built in Python, capable of performing basic arithmetic operations like addition, subtraction, multiplication, and division. The project is designed to help new Python learners understand key concepts such as:

  • Taking user input
  • Using conditional statements (if-elif-else)
  • Performing mathematical operations
  • Displaying formatted output

The code is well-commented, easy to read, and can be extended to include advanced features such as exponentiation, square root, or a graphical user interface (GUI).

Features

  • ✅ Addition
  • ✅ Subtraction
  • ✅ Multiplication
  • ✅ Division
  • ✅ Error handling for invalid input
